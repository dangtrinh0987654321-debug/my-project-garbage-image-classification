# my-project-garbage-image-classification
Đề tài tập trung xây dựng mô hình học máy gán nhãn rác thải từ hình ảnh nhằm phân loại rác thành các nhóm như tái chế, không tái chế và hữu cơ. Mô hình được huấn luyện trên tập ảnh đã gán nhãn, hướng đến hỗ trợ phân loại rác tự động và nâng cao hiệu quả tái chế.
